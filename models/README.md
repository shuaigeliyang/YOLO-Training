## 使用方法

```python
from ultralytics import YOLO

# 加载模型
model = YOLO('models/等级3_5000次.pt')

# 进行预测
results = model('path/to/image.jpg')

# 显示结果
results[0].show()
```

## 模型对比

不同等级和训练次数的模型可能有不同的性能表现。建议根据具体应用场景选择合适的模型。

**一般建议：**
- 快速检测：使用训练次数较少的模型（如 500次、1000次）
- 高精度检测：使用训练 5000 次的模型
